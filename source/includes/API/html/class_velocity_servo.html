Description détaillée

Objet permettant de controller la vitesse d'un servomoteur.

class VelocityServo est un outil permettant de déplacer un servomoteur vers une position à une vitesse donnée. Il utilise la classe Servo.h et PeriodicProcess Par défaut la vitesse du servomoteur est de 360 degrés/secondes.
