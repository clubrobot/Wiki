Description détaillée

Utilitaire pour gérer le temps dans vos programmes Arduino.

class Clock

Auteur
Ulysse Darmet est un outil permettant à vos programmes d'utiliser plus simplement la méthode micros() de <Arduino.h>. Cette objet vous permettera de mesurer le temps écoulé depuis le dernier appel de la méthode Clock::restart .
