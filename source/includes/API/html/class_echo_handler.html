Description détaillée

Gère le pin de reception du capteur.

class EchoHandler Cette classe est uniquement utilisée par la classe UltrasonicSensor et sert à gérer les émissions d'impulsion Par conséquent, il n'y a pas d'objet EchoHandler dans le .ino

Il existe un objet EchoHandler pour chacun des pins utilisables par l'arduino nano (Numérique 2 et 3) L'attribution s'effectue au moment dans la fonction attach() d'UltrasonicSensor.

Il est déconseillé d'utiliser deux capteurs sur le même EchoHandler
